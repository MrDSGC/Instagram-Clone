# API Endpoints

## HTML API

### Root

- `GET /` - loads React web app

## JSON API

### Users

- `POST /api/users`
- `PATCH /api/users`
- `GET /api/users/:id` (get profile page of user)

### Session

- `POST /api/session`
- `DELETE /api/session`

### Photos
- `GET /api//photos`
- `GET /api/photo/:id`
- `POST /api/users/photos`
- `PATCH /api/users/photos`

### Follows

- `POST /api/follows`
- `DELETE /api/follows`

### Likes

- `POST /api/likes`
- `DELETE /api/likes`
